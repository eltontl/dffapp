# DffApp

Aplicação Full Stack para gestão financeira pessoal, desenvolvida com arquitetura separada entre frontend e backend, utilizando tecnologias modernas do ecossistema JavaScript/TypeScript.

## 🚀 Tecnologias

### Frontend
- React
- Vite
- TypeScript

### Backend
- Node.js
- TypeScript

---

# 📁 Estrutura do Projeto

```text
dffapp/
│
├── backend/        # API e regras de negócio
│
├── frontend/       # Interface da aplicação
│
├── .gitignore
└── README.md
```

---

# ⚙️ Configuração do Ambiente

## Pré-requisitos

- Node.js 20+
- NPM 10+
- Git

---

# 📦 Instalação

## Clone o repositório

```bash
git clone <url-do-repositorio>
```

---

## Backend

```bash
cd backend
npm install
```

### Executar ambiente de desenvolvimento

```bash
npm run dev
```

---

## Frontend

```bash
cd frontend
npm install
```

### Executar ambiente de desenvolvimento

```bash
npm run dev
```
